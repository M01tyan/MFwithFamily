# Money Forward with Family (demo)
## セットアップ手順
1. リポジトリのコピー  
`$git clone https://github.com/M01tyan/MFwithFamily.git && cd MFwithFamily/`
1. Heroku CLIのインストール  
`$brew tap heroku/brew && brew install heroku`
1. ローカル環境でHerokuの起動  
`$heroku local web`
1. ブラウザを開く  
`localhost:5000/`
1. 下記のメールアドレスとパスワードでログイン
